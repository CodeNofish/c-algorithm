给定 n 种物品，每种物品都有重量 w i 和价值 v i ，其数量没有限制。
背包容量为 W ，求解在不超过背包容量的情况下如何放置物品，使背包中物品的价值之和最大。

和01背包区别，是无数量限制，0N背包

```c++
for (int i = 1; i <= n; i++)
    for (int j = 1; j<=W; j++)
        if (j < w[i])
            dp[i][j] = dp[i-1][j];
        else
            dp[i][j] = max(dp[i-1][j], dp[i-1][j-w[i]] + v[i]);
```
